# css

层叠样式表，负责页面的表现



# 设置样式的三种方式

+ 内联样式(行内样式)：标签上设置style属性

+ 内部样式表：head 使用style标签
+ 外部样式表：引入外部样式，通过link标签引入`<link rel="stylesheet" href="">`



# 选择器

选中页面中的指定元素

## 元素选择器

根据标签名来选择指定元素

## id选择器

根据元素的id属性值选择指定元素

#id属性值

## 类选择器

根据元素的class属性值选择一组元素

 .class属性值

一个元素可以指定多个class,中间逗号隔开

## 通配选择器

选中页面中的所有元素

*{}



## 交集选择器

选择同时复合多个条件的元素

```html
<style>
    /*选择div标签class为red(中间没有空格！)的标签颜色设置为红色*/
    div.red{
        color: red;
    }
</style>
<body>
    <div class="red">aaa</div>
    <p class="red">bbb</p>
</body>
```

## 并集选择器

同时选择多个元素

```html
<style>
    /*div和p(中间有逗号)标签颜色设置为红色*/
    div, p {
        color: red;
    }
</style>
<body>
    <div class="red">aaa</div>
    <p class="red">bbb</p>
</body>
```



## 子元素选择器

选中指定父元素的指定子元素

父元素  > 子元素

```html
<style>
    /*div下的span元素 颜色设置为红色*/
    div > span {
        color: red;
    }
</style>
<div>
    <p>
        被div包含的p元素
        <span>被p元素包含的span元素</span>
    </p>
    <span>被div的span元素</span>
</div>
```



## 后代元素选择器

选中指定元素内的指定后代元素

祖先元素 后代元素(中间有一个空格)

```html
<style>
    /*div下的所有span元素(所有后代，中间有个空格) 颜色设置为红色*/
    div span {
        color: red;
    }
</style>
```

## 兄弟选择器

选择下一个兄弟元素

前一个元素 + 后一个元素

```html
<style>
    /*p元素的下一个是span元素(中间是一个+号，选中p元素的下一个span元素) 颜色设置为红色*/
    p + span {
        color: red;
    }
</style>
```

```html
<style>
    /*p元素的下面的所有是span元素(选中所有是span的兄弟元素) 颜色设置为红色*/
    p ~ span {
        color: red;
    }
</style>
```

## 属性选择器

```html
<style>
    /*选中属性aaa为a 的元素颜色为红色*/
    [aaa=a] {
        color: red;
    }

    /*选中p元素并且有属性bbb 的元素颜色为蓝色*/
    p[bbb] {
        color: blue;
    }

    /*选中p元素并且属性有ccc但是值是以c开头 的元素颜色为黄色*/
    p[ccc^=c] {
        color: yellow;
    }

    /*选中p元素并且属性有ccc但是值是以c结尾 的元素颜色为绿色*/
    p[ccc$=c] {
        color: green;
    }

    /*选中p元素并且ccc属性的值中含有c 的元素背景颜色为红色*/
    p[ccc*=c] {
        background-color: red;
    }
</style>
<body>
    <div>
        <p aaa="a">aaa</p>
        <p bbb="b">bbb</p>
        <p ccc="c1">ccc</p>
        <p ccc="1c">ccc</p>
    </div>
</body>
```

## 伪类选择器

+ 一个:  例如 `:active`

+ 用来描述一个元素的特殊状态(不存在的类，特殊的类)

+ `:first-child` : 选择第一个子元素

```html
<style>
    /*将第一个li设置为红色*/
    ul > li:first-child {
        color: red;
    }
</style>
<body>
    <ul>
        <li>1</li>
        <li>2</li>
        <li>3</li>
        <li>4</li>
        <li>5</li>
    </ul>
</body>
```

+ `:last-child` : 选择最后一个子元素
+ `:nth-child(n)` : 选择第n个子元素 
  + 如果填n 就是0-n就是所有元素
  + 如果填2n或even 选择所有偶数子元素
  + 如果填2n+1或odd 选择所有的奇数子元素

+ `first-of-type、last-of-type、nth-of-type` 和上面的类似，但是要求同类型元素

```html
<style>
    /*如果第一个是其他标签就不成立，所有我们要求同类型*/
    /*上面的伪类都是根据所有的子元素进行排序的*/
    ul > li:first-of-type {
        color: red;
    }
</style>
</head>
<ul>
    <span>0</span>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>
```

+ `:not()` 否定伪类，将符合条件的元素从选择器中去除

```css
/*除了第二个元素其他都变红*/
ul > li:not(:nth-child(2)) {
    color: red;
}
```



## 超链接的伪类

```css
/*没有访问过的超链接颜色变为红色(只有a标签可用)*/
a:link {
    color: red;
}

/*访问过的超链接颜色为绿色(只有a标签可用)*/
a:visited {
    color: green;
}

/*悬停在a标签的时候颜色变蓝(所有标签可用)*/
a:hover {
    color: blue;
}

/*鼠标点击时变为黄色(所有标签可用)*/
a:active {
    color: yellow;
}
```



## 伪元素选择器

+ 两个:: 例如  `::after`

+ 描述页面中特殊的但是并不存在的元素

+ `::first-letter` 表示第一个字符

```html
<style>
    /*第一个字符，首字母下沉*/
    p::first-letter{
        font-size: 20px;
    }
</style>
<body>
    <p>hello word!</p>
</body>
```

+ `::first-line` 表示第一行
+ `::selection` 选中的元素

```css
/*选择的字符背景颜色变成黄色*/
p::selection{
    background: yellow;
}
```

+ `::before` 元素开始的位置
+ `::after` 元素最后的位置

```html
<style>
    p::before{
        content: "《";
        color: red;
    }
    p::after{
        content: "》";
        color: red;
    }
</style>
<body>
    <p>钢铁是怎么炼成的！</p>
</body>
```



# 选择器的优先级

内联样式>id选中器>类和伪类选择器>元素选择器>*(通配选择器)>继承的样式

也可以在样式后加上`!important` 让当前样式达到最高优先级



# 长度单位

像素(px)：屏幕有一个一个的像素点组成的

em: 相对于元素的字体大小来计算的   1em = 1font-size

rem: 相对于根元素(html)的字体大小来计算的



# 盒子模型

## 内容区

width和height的区域

## 边框

border的样式:

+ 宽度 border-width
+ 颜色 border-color
+ 样式 border-style

## 内边距(padding)

内容区和边框之间的距离

内边距的设置会影响盒子大小



## 外边距(margin)

不会影响盒子的可见框大小，但会影响盒子的位置



## 水平方向布局

水平方向设置了margin 会影响页面的布局

左外边距+左边框+左内边距+宽度+右内边距+右边框+右外边距 = 父元素的内容宽度

不够或超出会自行调整

左外边距、宽度、右外边距可以设置auto。这也是margin: 0 auto会自动居中的原理

## 垂直方向布局

父元素没有设置高，会被子元素挤出，如果设置了不会改变

## 子元素溢出父元素

使用overflow设置父元素如何处理溢出的子元素

+ visible：默认值，会溢出
+ hidden：隐藏溢出
+ scroll: 可滚动条显示
+ auto: 根据需要生成滚动条



## 外边距折叠

margin是表示盒子与盒子直接的距离，如果两个盒子在同一方法(互相的方向)都设置，那么会取其大值(绝对值)

如果子元素设置margin-top那么父元素也会被挤下来，这是因为子元素的传递给了父元素



## 行内元素

行内元素不支持设置宽高

行内元素可以设置padding,不会影响处置方向的页面布局

行内元素可以设置border,垂直方向不会影响布局

可以设置margin,垂直方向不会影响布局



## display: 

设置元素显示类型

+ inline: 设置为行内元素
+ block: 设置为块元素
+ inline-block: 设置为行内块元素，即可设置宽高也不会独占一行
+ table: 将元素设置为表格
+ none: 元素不在页面中显示(不在页面中占据位置)

## visibility: 

用来设置元素的显示状态

+ visible：默认值。元素正常显示
+ hidden: 元素不在页面中显示(依然占据位置)



## 盒子的尺寸

默认情况下，盒子的可见框大小内容区+内边距+边框

box-sizing  用来设置盒子尺寸的计算方式 (设置width和height的作用)

+ content-box 默认值，宽度和高度用来设置内容区的大小
+ border-box 宽度和高度用来设置整个盒子的可见框的大小



# 轮廓阴影和圆角

+ outline 用来设置元素的轮廓线，用法和border一样，轮廓不会影响可见框的大小

+ box-shadow  用来设置元素的阴影效果，也不会影响页面的布局
  + `box-shadow: 10px 10px 10px black` 前两个值是方向，第三个值是模糊半径

+ border-radius 用来设置圆角
  + `border-radius: 10px` 圆角的半径为10px
